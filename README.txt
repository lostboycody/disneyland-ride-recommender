To run the Disneyland recommender application:

Visit http://kivy.org/#download and download Kivy 2.7 for your machine.

Once downloaded, extract the download and navigate to the folder it was extracted to.

This program was only tested on a Windows 10 machine, but should work for any version of Windows 7+.

On windows, navigate to C:\Users\<Your Name>\Downloads\Kivy-1.9.0-py2.7-win32-x64

Click on the batch file named "kivy-2.7" to open a command prompt with Kivy bootstrapped. Once in this command prompt, run App.py by typing "python App.py" if you have python as a PATH in your pc's environment.
